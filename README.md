# taro-generate-router-method

> Taro 封装路由方法。

## 安装

在 Taro 项目根目录下安装

```bash
$ npm i taro-generate-router-method --save
```

## 使用

### 引入插件


修改项目 `config/index.js` 中的 plugins 配置为如下

```js
const config = {
  ...
  plugins: [
    ...其余插件
    'taro-generate-router-method'
  ]
  ...
}
```

```
执行 taro router-methods

```
